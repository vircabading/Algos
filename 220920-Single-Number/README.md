# Single Number Algorithm

# Description

Given a non-empty array of integers (nums), every element appears twice except for one.  Find that single one.

# Language

JavaScript

# Origin

LeetCode

# Examples

```
nums = [2, 2, 1]
result = 1
```

```
nums = [4, 1, 2, 1, 2]
result = 4
```

```
nums = [1]
result = 1
```
